## To run

0. Install deps `npm install`
1. Make sure Redis, and Postgres running
2. Configure the .env file for postres in `/.env` and for redis in `./apps/redis/.env`
3. Apply migration in root directory of project by using command `npx prisma migrate dev`
4. Spin up the service </br>
   a. Gateway `npx nest start gateway --watch`</br>
   b. Gateway `npx nest start create --watch`</br>
   c. Gateway `npx nest start update --watch`</br>
   d. Gateway `npx nest start read --watch`</br>
   e. Gateway `npx nest start delete --watch`</br>
   f. Gateway `npx nest start redis --watch`</br>
5. The Gateway is running on `localhost:3000` and the docs can be seen in `localhost:3000/api/docs`

## About Test

1. Gateway, hidden all the service, and can be only accessed via gateway that provide http service
2. Create Service, create user
3. Read Service, find all user, find user by id, find user by identity number, find userr by account number
4. Update Service, update user
5. Delete Service, delete user
6. Redis Service, set, get, delete

((find-all api, in Read Service is implementing caching system, automaticly updated after update or create))

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
